Teste de performance

Pagina dedicada a medir o desempenho do algoritmo. Ao inserir os dados e selecionar um algoritmo, um script irá executá-lo 30 vezes, calculando seu desempenho médio e possibilitando uma análise geral do comportamento para a série de dados.

Atenção! O processamento dos dados é lento, principalmente para funções que não podem ser encontradas. Se for testar alguma série de dados, é recomendado realizar um teste na página inicial para certificar-se de que a equação pode ser encontrada.
