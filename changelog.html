Registro de updates

O desenvolvedor não garante que os itens na lista de afazeres serão implementados, mas, por questão de organização, todas as possíveis ideias são colocadas anotadas lá, e são detalhes técnicos da implementação do projeto. Caso canceladas, serão riscadas, e conforme forem sendo resolvidas, serão movidas para a lista de updates.

Para acompanhar cada detalhe do que foi feito, visite o GitHub do projeto. O repositório é sempre atualizado ao mesmo tempo que este quadro de updates.
