Funções incompatíveis

Devido à limitações da estrutura de dados (IT) utilizada para compor as expressões, existem funções que nunca serão criadas. Todos os algoritmos se baseiam nessa estrutura de dados, e suas expressões são compostas somando várias estruturas IT.

Cada estrutura IT guarda dentro dela um vetor de n expoentes e uma operação. Então, para calcular seu valor, a estrutura recebe um vetor que corresponde a uma linha de entrada e eleva todas as variáveis da linha de entrada aos seus respectivos expoentes. Após isso, multiplica todas as variáveis da linha de entrada e, por fim, resolve a operação com o valor obtido.

Gerador de dados (à partir de expressões)

Caso queira gerar rapidamente uma tabela de dados para usar no site, basta escrever uma expressão na caixa abaixo. Ao usar o gerador lembre-se de definir um índice para as variáveis, começando do 0 (x0, x1, x2, ...). O script gera até 40 linhas, e suporta expressões com até 9 variáveis. O gerador segue a sintaxe do javaScript, e as funções matemáticas devem ser usadas de acordo com a biblioteca de funções Math (ex: Math.sin(x2) + Math.sqrt(x1) + Math.abs(x3)).
