Obs:

Atenção! As tabelas para a equação de trabalho são diferentes (perceba que cos(Θ) é diferente de Θ)!

- A página principal aceita separações por vírgulas, tabulações e espaços em branco, e ignora qualquer entrada que não seja um número.

- As linhas de entrada das funções são geradas assim que a página carrega, portanto uma entrada que você tenha usado antes talvez seja diferente, mas ainda vem da mesma expressão.

- funções trigonométricas usam radianos.

- O padrão numérico utilizado é o americano, com casas decimais separadas por ponto.

- O ruído adiciona um erro de até 0.3 nos dados aleatóriamente.

Gerador de dados (à partir de expressões)

Caso queira gerar rapidamente uma tabela de dados para usar no site, basta escrever uma expressão na caixa abaixo. Ao usar o gerador lembre-se de definir um índice para as variáveis, começando do 0 (x0, x1, x2, ...). O script gera até 40 linhas, e suporta expressões com até 9 variáveis. O gerador segue a sintaxe do javaScript, e as funções matemáticas devem ser usadas de acordo com a biblioteca de funções Math (ex: Math.sin(x2) + Math.sqrt(x1) + Math.abs(x3)).
