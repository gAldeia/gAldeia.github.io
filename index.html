Regressão Simbólica usando técnicas de Programação Genética

Página do projeto de iniciação científica que aplica técnicas de programação genética no método de regressão simbólica.

Sobre

Regressão simbólica é uma técnica que consiste na criação e manipulação de expressões matemáticas a fim de encontrar um modelo que seja capaz de descrever suficientemente bem uma série de pontos. As expressões são criadas arbitráriamente e são avaliadas.

Sua aplicação se torna muito mais poderosa com técnicas computacionais, já que o computador pode realizar muito mais rápido operações do que um humano. Assim, com a programação genética, podemos realizar uma busca ampla pelo universo de expressões matemáticas e tentar encontrar o modelo que descreve a série de pontos.

Resultado

Conforme a evolução for ocorrendo, informações sobre a população e a melhor resposta que conseguimos serão mostradas e atualizadas no quadro abaixo.

O que se espera é encontrar a equação que deu origem aos dados de entrada. Se os dados vieram de uma equação, espera-se encontrar a própria equação (ou uma muito próxima dela).

Para dados que vieram de um experimento, por exemplo, espera-se encontrar um modelo matemático que descreva os pontos da entrada.

O resultado obtido foi o esperado?

A evolução é muito influenciada pelos melhores indivíduos das primeiras gerações. Casos infelizes onde uma expressão insatisfatória (distante da expressão que rege os dados de entrada) guie esta primeira etapa podem acontecer. Com a primeira geração sendo criada aleatóriamente, o rumo da evolução pode acabar sendo erroneamente direcionado. Se o resultado obtido não parece, a princípio, correto, tente executar o algoritmo mais uma vez. Este fator aleatório pode ter uma influência minimizada quando vários testes são realizados, ou quando a população é grande.
