O problema de regressão

Regressão é um processo estatístico que tem como objetivo encontrar uma relação entre variáveis. Dado um conjunto onde cada elemento é um conjunto de variáveis, Então a missão de um método de regressão, para um conjunto de variáveis, é encontrar uma função.

Regressão Linear: Na regressão linear, o palpite é de que a função que se ajusta ao conjunto de variáveis é uma composição linear de parâmetros. numa regressão linear simples ainda pode existir um termo independente. Os algoritmos do site implementam uma regressão linear, onde cada algoritmo tem sua particularidade.

A estrutura de dados IT

Devido à limitações da estrutura de dados (IT) utilizada para compor as expressões, existem funções que nunca serão criadas. Todos os algoritmos se baseiam nessa estrutura de dados, e suas expressões são compostas somando várias estruturas IT.

Cada estrutura IT guarda dentro dela um vetor de n expoentes e uma operação. Então, para calcular seu valor, a estrutura recebe um vetor que corresponde a uma linha de entrada e eleva todas as variáveis da linha de entrada aos seus respectivos expoentes. Após isso, multiplica todas as variáveis da linha de entrada e, por fim, resolve a operação com o valor obtido.

O algoritmo SymTree

Este algoritmo inicia sua busca a partir de uma solução representando uma regressão linear. A cada iteração ele aplica as operações de interação entre as variáveis e transformação, gerando funções incrementalmente mais complexas. Esse algoritmo é executado por oito iterações. O algoritmo IT-LS

O algoritmo funciona criando uma população inicial de expressões aleatórias e selecionando a melhor dentre elas. Após isso, executa uma busca local na equação, mudando ou as funções não-lineares ou os expoentes da equação, repetindo o processo até que não exista uma modificação que melhore ainda mais o score da equação. Esse algoritmo é executado até no máximo 50 iterações.

O algoritmo IT-ES

O algoritmo funciona criando uma população de expressões aleatórias, e então executa um algoritmo de Estratégia Evolutiva ES-(mu, lambda). Esse algoritmo foi executado apenas com mutação, mu=150,lambda=45 e 150 iterações
